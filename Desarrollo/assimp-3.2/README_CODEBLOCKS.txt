El proyecto hay que generarlo cada uno con su path con CMAKE
Instalar CMAKE GUI para windows:
	Source code: la carpeta de assimp-3.2 (ruta absoluta)
	Build the binaries: la carpeta assimp-3.2/build (ruta absoluta)
	Botón de Configure
	Desactivar ASSIMP_BUILD_TESTS
	Botón de Generate

El proyecto para codeblocks se crea en build/Assimp.cbp
Una vez compilado, el .a y la dll en build/code
