Sistema de gestión de eventos (Trigger System/Event Manager)

Source Code in ..\GitHub\lab21\Desarrollo\game\src\classes:

- Trigger
- TriggerDoor
- TriggerGenerator

Trigger and TriggerSystem are in Alfa. Triggers are the responsible of activate the doors and generators. In the future Shoots, Sounds, Perception, Enemies using Triggers and other things will be added to the Trigger System. 

To active current Triggers, the player can press Space to do an action like open/close a door or activate a generator while standing inside one Trigger.
