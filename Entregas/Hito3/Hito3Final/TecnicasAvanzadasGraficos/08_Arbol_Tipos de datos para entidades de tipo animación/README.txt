Este entregable contiene los archivos necesarios para la creación de tipos animados en el árbol de la escena.

Los archivos EAnimation contienen las variables y métodos para definir los tipos animados.

Los tipos animados se crean en nuestro motor gráfico en TAGEngine.

Los archivos TAGEngine.h y TAGEngine.cpp has sido recortados para mostrar sólo el código utilizado para realizar los tipos animados, facilitando así su localización.
