Se definen las clases necesarias para los objetos que crean el árbol del motor.

GraphicNode -> Todos los elementos del árbol son GraphicNode. Pueden tener un o ningún padre y 0,n hijos. Puede contener un objeto tipo entidad.
Entity 		-> Clase base para las entidades.

Se adjuntan las declaraciones y definiciones de las clases en el directorio source.

En el juego (directorio Hito3Final/Lab21Game) se puede ver la ejecución de estas clases y en source el proyecto en CodeBlocks.
